# Coast FIRE Calculator

## 💰 About

The Coast FIRE Calculator is a powerful yet simple web tool designed to help users estimate their path to financial independence. Built with a modern Neumorphism (Soft UI) design, it offers a clean, intuitive, and distraction-free experience.

Coast FIRE (Financial Independence, Retire Early) is a strategy where you accumulate enough investments early so they can grow independently over time, allowing you to reduce or stop contributions while still reaching retirement goals. This calculator helps you visualize that journey by projecting your future investment value.

---

## ✨ Key Features

- Accurate future value and growth projections
- Clean Neumorphic (Soft UI) interface
- Fully responsive (mobile-first design)
- Simple and fast calculations
- No data storage or tracking
- Lightweight and optimized for performance

## 📌 Usage

1. Enter your current savings  
2. Add your monthly contribution  
3. Input expected annual return (%)
